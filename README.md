# 👾 Alura Geek

O Alura Geek é um e-commerce onde é possível adicionar qualquer item da escolha do usuário, incluindo seu nome, valor do produto e imagem via URL.

## ⚙️ Tecnologias utilizadas no projeto

* HTML
* CSS
* Javascript
* Json-server
